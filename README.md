# Netflix Clone - React & Tailwind

A beautiful Netflix clone focusing on UI/UX, built with Modern React 18, TypeScript, and Tailwind CSS.

## Features
- 🍿 **Dynamic Banner**: Featured movie banner with gradients.
- 📜 **Horizontal Rows**: Smooth-scrolling movie categories.
- 🎨 **Netflix UI**: High-fidelity reproduction of Netflix's dark theme.
- 📱 **Fully Responsive**: Works on mobile, tablet, and desktop.
- ✨ **Framer Motion**: Subtle animations on hover and scroll.

## Getting Started
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`
